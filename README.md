# IdeaLync - internal bot for Solync

## Getting Started

> [!IMPORTANT]
> Please make sure to install [Bun](https://bun.sh/). This project is optimized for Bun.
> Also install [Prettier](https://prettier.io/docs/editors).

To begin development and testing locally, please follow these steps in your terminal of choice:

1. Clone the repo by running `git clone https://github.com/SolyncSoftware/IdeaLync.git`.
2. Go inside the newly cloned folder (`cd IdeaLync`).
3. Run `bun install` to install the packages.
4. Great! Now run `I FORGOT THE COMMAND PLEASE REPLACE THIS` to run the development server.
